The purpose of this Demo project is to showcase how you could use the PerformanceLabs framework. There is some initial settings to be done in the SetupTests.cs file. For the purposes of this project, the DemoReport.html file will be created in ..\PerformanceReport directory. Whatever the framework has to be offer will be presented in this demo project. You can run this project by clicking the RunDemoProject.bat in the HelperUtilities directory.
